# Suggested Commands
- Install deps: `pnpm install` (pnpm lockfile present; Node >= 22.12).
- Dev server: `pnpm dev` for hot-reload Astro.
- Lint/type/accessibility check: `pnpm lint` (runs `astro check`).
- Build static site + search index: `pnpm build` (Astro build then Pagefind over `dist`).
- Preview built site: `pnpm preview`.
- Optional: `pnpm astro ...` to run arbitrary Astro CLI tasks.
